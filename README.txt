*****************************************************************************************************
IMPORTANT INSTALLATION INSTRUCTIONS, READ BEFORE USING
*****************************************************************************************************
1. Download and unzip this project, or clone the repo, into your WordPress project's themes directory.
2. Change the name of the theme's directory from ndnation to themename (themename is what ever you name your theme, obviously not themename).
3. Navigate the the themes lib/admin directory.
4. Change the name of the file: class.ndnation-update-checker.php to: class.themename-update-checker.php.
5. Open your project in Sublime Text.
6. Perform a search and replace on the theme's directory.
7. Search for ndnation and replace it with themename.
8. If you're using a case sensitive search also search for NDNation and replace it with Theme Name.
9. Delete the .git directory from your theme's directory.
10. Replace screenshot.png with an image of your theme.
11. Commit your new theme to your project's repo. You MUST NOT commit your changes to the ndnation repo.
12. Congratulations! You're finished.

********************************************************************

SUSHI INSTALLATION INSTRUCTIONS, INCASE YOU MISSED IT THE FIRST TIME

********************************************************************
1. Download the zip file.
2. Search and replace all instances of ndnation with your_theme_name.
3. Search and replace all instances of NDNation with Your_Theme_name.
4. Update style.css info with your theme name and version.
5. Update the theme updater files with your theme version and info.
6. Replace screenshot.png with and image of your theme.
7. Keep it DRY!! Templates go in the template folder, and template parts go in the parts folder.
